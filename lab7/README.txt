Lab 7: Shading

Run:
..\\.venv\\Scripts\\python.exe lab7.py

Controls:
1 - Task A
2 - Task B
3 - Task V
M - switch Gouraud/Phong (only in Task B)
Space - pause/resume

What is implemented:
- Task A: one low-poly sphere, one point light, Gouraud shading only.
- Task B: two objects (sphere and torus), ambient lighting, real-time switch Gouraud/Phong.
- Task V: several objects (sphere, torus, cube), two point lights with different colors,
  Phong specular highlights, and simple projected shadows on a plane via shadow matrix.
